## Задание 1. Условные конструкции

### Описание

С помощью функции `date()` формируется таблица расписания, зависящая от текущего дня недели. Функция `date('N')` возвращает номер дня недели: 1 — понедельник, 7 — воскресенье.

Правила формирования расписания:
- **John Styles** — работает в понедельник, среду и пятницу (дни 1, 3, 5) с `8:00 до 12:00`. В остальные дни — `Нерабочий день`.
- **Jane Doe** — работает во вторник, четверг и субботу (дни 2, 4, 6) с `12:00 до 16:00`. В остальные дни — `Нерабочий день`.

### Реализация

```php
<?php
$day = date('N');


if ($day == 1 || $day == 3 || $day == 5) {
    $johnSchedule = "8:00 - 12:00";
} else {
    $johnSchedule = "Нерабочий день";
}


if ($day == 2 || $day == 4 || $day == 6) {
    $janeSchedule = "12:00 - 16:00";
} else {
    $janeSchedule = "Нерабочий день";
}
?>

<table border="1">
    <tr>
        <th>№</th>
        <th>Фамилия Имя</th>
        <th>График работы</th>
    </tr>
    <tr>
        <td>1</td>
        <td>John Styles</td>
        <td><?= $johnSchedule ?></td>
    </tr>
    <tr>
        <td>2</td>
        <td>Jane Doe</td>
        <td><?= $janeSchedule ?></td>
    </tr>
</table>
```

### Результат

| № | Фамилия Имя | График работы |
|---|---|---|
| 1 | John Styles | 8:00 - 12:00 *(пн, ср, пт)* / Нерабочий день |
| 2 | Jane Doe | 12:00 - 16:00 *(вт, чт, сб)* / Нерабочий день |

## Задание 2. Циклы

### 2.1. Цикл `for` с выводом промежуточных значений

```php
<?php
$a = 0;
$b = 0;

for ($i = 0; $i <= 5; $i++) {
    $a += 10;
    $b += 5;
    echo "Шаг $i: a = $a, b = $b <br>";
}

echo "Конец for-цикла!";
```

**Вывод:**
```
Шаг 0: a = 10, b = 5
Шаг 1: a = 20, b = 10
Шаг 2: a = 30, b = 15
Шаг 3: a = 40, b = 20
Шаг 4: a = 50, b = 25
Шаг 5: a = 60, b = 30
Конец for-цикла!
```

---

### 2.2. Цикл `while`

```php
<?php
$a = 0;
$b = 0;
$i = 0;

while ($i <= 5) {
    $a += 10;
    $b += 5;
    echo "Шаг $i: a = $a, b = $b <br>";
    $i++;
}

echo "Конец while-цикла!";
```

**Вывод:** аналогичен циклу `for`.

---

### 2.3. Цикл `do-while`

```php
<?php
$a = 0;
$b = 0;
$i = 0;

do {
    $a += 10;
    $b += 5;
    echo "Шаг $i: a = $a, b = $b <br>";
    $i++;
} while ($i <= 5);

echo "Конец do-while-цикла!";
```

**Вывод:** аналогичен циклу `for`.

## Контрольные вопросы

### 1. В чём разница между циклами `for`, `while` и `do-while`? В каких случаях лучше использовать каждый из них?

| Цикл | Проверка условия | Когда использовать |
|---|---|---|
| `for` | До выполнения тела | Когда заранее известно количество итераций |
| `while` | До выполнения тела | Когда количество итераций заранее неизвестно, тело может не выполниться ни разу |
| `do-while` | После выполнения тела | Когда тело цикла должно выполниться **хотя бы один раз** |

---

### 2. Как работает тернарный оператор `? :` в PHP?

Тернарный оператор — это сокращённая форма конструкции `if-else`. Синтаксис:

```php
$result = (условие) ? значение_если_истина : значение_если_ложь;
```

**Пример:**

```php
$day = date('N');
$johnSchedule = ($day == 1 || $day == 3 || $day == 5) ? "8:00 - 12:00" : "Нерабочий день";
```

Это эквивалентно:

```php
if ($day == 1 || $day == 3 || $day == 5) {
    $johnSchedule = "8:00 - 12:00";
} else {
    $johnSchedule = "Нерабочий день";
}
```

Тернарный оператор удобен для коротких условий, повышает компактность кода.

---

### 3. Что произойдёт, если в `do-while` поставить условие, которое изначально ложно?

Тело цикла `do-while` **выполнится ровно один раз**, после чего проверяется условие. Поскольку условие ложно, цикл завершается.

**Пример:**

```php
$i = 10;
do {
    echo "Это выполнится один раз, i = $i";
    $i++;
} while ($i <= 5); // условие изначально ложно (10 <= 5 — false)
```

**Вывод:**
```
Это выполнится один раз, i = 10
```

Именно этим `do-while` отличается от `while`: цикл `while` при изначально ложном условии не выполнится ни разу.
